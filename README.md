デモ動画

BPMに合わせてマーカーを挿入することができるPremiere Proエクステンションです。

再生バーの位置から、任意の秒、指定したBPMでマーカー配置できます。

是非音MAD等にご活用ください。

このプロジェクトは[Bolt CEP](https://hyperbrew.co/resources/bolt-cep)の力によって作成されました
